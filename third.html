Can You Learn IP Addresses?

IP Addresses are in every eletronic device that is able to connect to a network (WiFi). These addresses permit different devices to communicate with each other. Wihtout them you are pretty much useless with you want web access. I made this little test so you can understand how IP addresses are assigned. Goodluck!!!

Some Important Patterns

  • 1010 0000 = 150
  • 0000 1000 = 16
  • 0100 0000 = 64
  • 0100 0020 = 66
  • 0000 0100 = 8
